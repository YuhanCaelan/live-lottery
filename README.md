# live-lottery

一个用于晚会、大屏幕和现场活动的静态抽奖页面，支持座位抽取、轮次设置、范围排除、背景与音乐自定义，并会在浏览器本地保存设置。

## 功能

- 大屏抽奖展示，适合现场投屏使用
- 支持设置抽奖轮数和每轮人数
- 支持一层、二层座位参与控制
- 支持按排、列快速排除座位范围
- 支持自定义背景图片和 BGM
- 支持抽奖结果展示、重置和本地记忆配置

## 文件结构

- index.html：页面结构
- style.css：界面样式与动效
- script.js：抽奖逻辑与交互

## 使用方式

1. 直接用浏览器打开 index.html。
2. 点击“设置”调整抽奖轮数、每轮人数和座位排除范围。
3. 可上传背景图片或音乐文件来自定义舞台效果。
4. 点击“开始抽奖”开始，点击“停止抽奖”锁定结果。
5. 抽完后可查看全部结果，或点击“重置抽奖”重新开始。

## 浏览器要求

- 建议使用现代桌面浏览器打开
- 本地存储用于保存设置，清理浏览器数据后会恢复默认状态

## 部署提示

- 这是静态前端项目，可直接部署到 GitHub Pages、Nginx 或任意静态托管服务
- 如果需要默认背景图，可以在项目中补充 assets/bg.jpg
